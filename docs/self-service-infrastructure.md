# Self-Service Infrastructure

## What is Self-Service Infrastructure?

Self-service infrastructure allows developers to provision and deploy resources without opening tickets or waiting for manual intervention.

The platform provides reusable modules and automated workflows that enforce security, governance, and operational standards.

---

## Benefits

- Faster delivery
- Consistent infrastructure
- Improved security
- Reduced operational burden
- Better developer experience

---

## Platform Team Responsibility

The platform team builds reusable infrastructure modules.

Developers consume those modules through approved Golden Paths.

---

## Examples

Developers should be able to:

- Deploy applications
- Request namespaces
- Access monitoring
- Access logs
- Use approved infrastructure

without requiring platform team assistance.

---

## Goal

Reduce manual infrastructure requests and enable self-service cloud operations.
